Rat-Maze Game

A puzzle/game made in Java to compete with other developers. The "rat" communicates with other players' mazes over the server using TCP/IP and solves the randomly generated maze. The "rat" uses the Breath First Search algorithm to solve the maze in optimal time.

Round Robin Scheduler

A scheduler made in Java which creates a schedule for the teams based on the Round-Robin algorithm. The schedule can take any number of teams and be limited by date. Also includes a backend database system to store scores/results.

Currency Converter

A realtime currency converter built in Java. It gets the latest currency rates via REST/JSON and parses them using GSON. Has options for displaying current rates and converting to different currencies. Currently supports over 15 currencies.

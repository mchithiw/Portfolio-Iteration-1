Ikra

Ikra allows users to read verses of the Quran by a specified Ayat, a complete Sura, or by Keywords. I initially made this because a few friends and I got tired of skimming through the Quran to find a specific topic and wanted a keyword search to provide faster results for the specific topics we were interested in.

Currencer

A Currency Converter that gets the latest currency data for over 165 countries and allows free conversion between them. The rates are updated very 10 minutes to accurately match real-time changes. The same application was also ported as a Java App, using the GSON library to parse the JSON from the API calls.

Sockets

Sockets is a turn based game made in Java to compete with fellow classmates. The concept of the game somewhat resembles "Battleship". Each player has a rat and a maze, and the rat tries to solve other people's mazes in the most optimal way. The rats communicate with the maze over the server using TCP/IP Socket communication, hence the name.

Robin

A scheduling system written in Java to properly create scheules and store scores for games/tournaments, which implemented the Round-Robin Scheduling Algorithm. The system involved all aspects of the process, including the entry of teams, scheduling, scoring, time management, and UI.
